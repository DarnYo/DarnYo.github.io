A game of conveyors

  • Solo project, developed as part of my coursework
  • Developed in Unreal Engine
  • Modelled a 3d asset using 3ds Max
  • Tasked with building a sidescroller puzzle game
  • Programmed the mechanics using Unreal's blueprints
  • Designed a level with escalating difficulty, exploring the design space created by the simple conveyor belt mechanic.

A game of conveyors was developed as part of my coursework on Computer Science for Games. The task was to create one 3d modelled asset using 3ds Max, animate it, and import it into Unreal Engine to create a puzzle prototype.

For my asset, I modelled a conveyor belt. The asset is actually made of separate segments, allowing the middle segment to be tiled in order to have any length of conveyor belt. The animation is done by moving the belt slightly, then teleporting it back into the starting place, creating the illusion of continuous movement.

In Unreal, the conveyor is put together using a construction script, which allows me to change the length via editor. The puzzles are built around having crates be pushed around by the conveyors and around changing the movement direction of the conveyors with a button press.
